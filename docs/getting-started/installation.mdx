## Requirements

- Node.js 20+ (LTS recommended)
- pnpm or npm

## Install the CLI

Use the CLI for the fastest project setup:

<CodeGroup>
  ```bash pnpm theme={null}
  pnpm add -g @fentaris/cli
  ```

  ```bash npm theme={null}
  npm i -g @fentaris/cli
  ```
</CodeGroup>

## Install @fentaris/core directly

Use the core package directly when embedding Fentaris in an existing TypeScript project:

<CodeGroup>
  ```bash pnpm theme={null}
  pnpm add @fentaris/core
  ```

  ```bash npm theme={null}
  npm i @fentaris/core
  ```
</CodeGroup>

## Verify installation

Create a quick file and import a Fentaris symbol:

```ts theme={null}
import { McpProxy } from "@fentaris/core";

console.log(typeof McpProxy);
```

If your build succeeds, Fentaris is installed.
