## Requirements

- Node.js 20+ (LTS recommended)
- pnpm or npm

## Install the CLI

Use the CLI for the fastest project setup:

<CodeGroup>
  ```bash pnpm theme={null}
  pnpm add -g @panther/cli
  ```

  ```bash npm theme={null}
  npm i -g @panther/cli
  ```
</CodeGroup>

## Install @panther/core directly

Use the core package directly when embedding Panther in an existing TypeScript project:

<CodeGroup>
  ```bash pnpm theme={null}
  pnpm add @panther/core
  ```

  ```bash npm theme={null}
  npm i @panther/core
  ```
</CodeGroup>

## Verify installation

Create a quick file and import a Panther symbol:

```ts theme={null}
import { McpProxy } from "@panther/core";

console.log(typeof McpProxy);
```

If your build succeeds, Panther is installed.
